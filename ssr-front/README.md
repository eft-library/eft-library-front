# 남은거

1. user quest 에 기존 데이터 없으면 에러남 조회를 못해서
2. 하루 한 번 주기로 search refresh dag 만들기
3. quest 연관 정보 전체 수정하기
4. 사용자 정보 페이지 만들기 - 닉네임 수정, 아이콘 수정, 작성 글 목록, 작성 댓글 목록, 개인 정보(가입 일, 포인트 등)
5. 로그인 콜백 구현 - 네이버, 카카오, 구글 연동 - 댓글 및 글 작성 가능, 악질 유저 차단 기능(댓글 및 글 작성 불가), 사용자 등급
6. 게시판 back 먼저 만들어 놓자 - 탭 분류(공략, 팁, 자유, 신고), 추천, 비추 - 댓글에도 추천 비추
7. 공지 사항
8. 패치 노트
