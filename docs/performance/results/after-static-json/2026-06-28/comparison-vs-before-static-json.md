## 해석

- 정적 JSON 전환 효과는 TBT 감소에서 가장 뚜렷합니다. `rank`, `price`, `hideout`, `quest`, `story`, `quest-detail`이 특히 좋아졌습니다.
- 전송량과 요청 수는 거의 동일합니다. 정적 JSON 전환은 네트워크 리소스 수를 크게 줄이기보다는 API 처리 비용과 서버 응답 안정성을 낮추는 쪽 효과가 큽니다.
- LCP는 이번 단일 측정에서 흔들림이 큽니다. `map-of-tarkov`의 `46.2s`는 이미지/외부 리소스 또는 Lighthouse 측정 변동 가능성이 높아서 별도 재측정 대상입니다.
- `live-map`은 요청 수와 전송량이 소폭 줄었고 CLS는 안정적입니다. 다만 마커가 많은 페이지의 스크롤/줌 체감 성능은 Lighthouse보다 브라우저 프로파일링으로 보는 편이 더 정확합니다.
